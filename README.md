# RaspberryPi4-Irrigation-Pompe
systeme d'irrigation avec raspberry pi 4 en utilisant de spompes 

# Objectif du projet
- Contrôler automatiquement l’irrigation des plantes à l’aide d’une pompe.
- Surveiller et ajuster le débit d’eau selon les besoins des plantes.
- Fournir un guide pratique pour le câblage et l’installation du système.
- Alerter automatiquement par email lors d'irrigation.

# Contenu
- RaspberryPi4-Irrigation-Pompe.pdf – Documentation complète du système.
- pompes_schema.png – Schéma du câblage des pompes et du Raspberry Pi.
- irrigation-avec-pompe.jpg – Photo illustrative du montage réel.
- Code_AlertEmail_irrigation – Code Python pour l’alerte par email d'activations et desactivations des pompes lors d'irrigation.
# Technologies
- Raspberry Pi
- Pompe d’irrigation 5v
- relais 
- batterie 9v
